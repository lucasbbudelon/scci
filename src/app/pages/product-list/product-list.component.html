Olá, bem vindo ao SCCI!

Sistema de compras inteligente para cegos. Este é um protótipo para validar uma ideia, que consiste em falar o nome e o valor dos produtos, conforme vão sendo somados em um processo de compra em supermercados.

Atenção para as orientações de uso:

  1. Peça para o cliente colocar o fone de ouvido;
  2. Explique que a partir de agora ele poderá ouvir/acompanhar os produtos e o seu valor;
  3. Para cada produto, insira o código do produto, informe a quantidade e precione enter (ou adicionar);
  4. Caso seja necessário remover um produto, precione o botão "remover";
  5. Ao final do pedito, clique em finalizar pedido.
